# Trippi Mobile (Expo)

Modern, engaging trip planning and budgeting app UI built with Expo and expo-router.

## Quickstart

```bash
npm install -g expo-cli # if not installed
cd mobile
npm install
npm run start
# Scan the QR with Expo Go on iOS/Android
```

## Scripts
- `npm start` – start Metro bundler (tunnel enabled)
- `npm run start:lan` – start on LAN
- `npm run ios` / `npm run android` – run native builds if desired
